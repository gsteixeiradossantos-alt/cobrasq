Gestão de usuários
Backup e importação de casos
Exportar: baixa um arquivo .json com todos os casos do sistema. Use para backup periódico.
Importar: insere casos a partir de um arquivo .json (do mesmo formato exportado). Casos duplicados (mesmo devedor + telefone + credor) são ignorados automaticamente.
Configuração de envio WhatsApp
wa.me (padrão): os botões "Enviar via WhatsApp" abrem o WhatsApp Web/App com a mensagem pré-pronta. A operadora confirma e envia manualmente.
Z-API: envia diretamente pela API do Z-API (sem abrir nada). Mais rápido, mas requer configuração de secrets no Supabase e a Edge Function enviar-whatsapp deployada.
⚠ Modo Z-API só funciona se a Edge Function enviar-whatsapp estiver deployada no Supabase com os secrets configurados (ZAPI_INSTANCE, ZAPI_TOKEN, ZAPI_CLIENT_TOKEN). Se não estiver, troque pra wa.me.
⏱ Prazos por etapa (cronômetro)
Tempo (em minutos) que cada etapa pode ficar em aberto antes do cronômetro virar amarelo (≥70%) ou vermelho (≥100%). Salvo localmente no navegador do gestor.
💳 Gateway de pagamento (Mercado Pago) — item #9
Habilite Mercado Pago como segundo gateway. Ao gerar acordo, o devedor verá o comparativo Boleto vs. Cartão MP com a taxa embutida.
Estrutura visual apenas — integração real ainda não implementada.

Sobre os papéis
Admin: vê todos os casos do sistema, pode criar/desativar usuários, editar qualquer caso.
Operador: vê apenas os casos que criou ou que estão atribuídos a ele.
Inativo: não consegue fazer login. Use para desligar alguém sem perder o histórico de casos dela.
